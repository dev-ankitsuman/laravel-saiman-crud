# Contributing to Laravel Saiman CRUD Generator

Thank you for considering contributing! Contributions are welcome and appreciated.

## Ground Rules

- Be respectful and follow our [Code of Conduct](CODE_OF_CONDUCT.md).
- Open an issue **before** submitting a large PR to discuss the change.
- Write tests for all new features and bug fixes.
- Follow PSR-12 and the existing code style.
